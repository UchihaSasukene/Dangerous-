# Dangerous

这是一个危化品管理系统的毕业设计项目，旨在提供一个全面的化学品管理解决方案。

## 项目概述

本项目是一个危化品管理系统，主要功能包括：
- 化学品信息管理
- 库存管理
- 使用记录跟踪
- 出库管理
- 存储记录
- 预警系统

## 技术栈

- 后端：Java (JDK 17)
- 前端：Vue 2.9.6
- HTTP 客户端：Axios@1.5.0
- UI 框架：Element-UI@2.15.14
- Node.js 环境：22.14

## 安装指南

### 后端

1. 使用 IntelliJ IDEA 或 VSCode 的 Java 运行方式启动项目
2. 确保数据库已正确配置并运行

### 前端

1. 进入 `vue-ui` 目录
2. 运行 `npm run dev` 启动开发服务器

## 功能特性

- 化学品信息管理：添加、编辑、删除化学品信息
- 库存监控：实时跟踪化学品库存量
- 使用记录：记录化学品的使用情况
- 出库管理：管理化学品的出库操作
- 存储记录：记录化学品的存储信息
- 预警系统：当库存量低于预警阈值时触发预警

## 注意事项

- 登录注册功能目前仅为演示实现，后续需要完善
- 项目还有一些功能尚未完全实现，等待后续开发完善
- 请确保数据库配置正确，并已执行初始化脚本
